# Ecosystem Overview

## Gems

- `ollama-client` — runtime kernel
- `ollama-openai` — OpenAI compatibility translation
- `ollama-stream` — advanced streaming runtime
- `ollama-observability` — telemetry and diagnostics
- `ollama-schema` — typed schema/repair runtime
- `ollama-testing` — mock/replay/failure simulation
- `ollama-rails` — Railtie + realtime Rails helpers
- `ollama-agent` — optional orchestration primitives
- `ollama-router` — routing and fallback logic

## Principles

- Small deterministic core
- Optional integrations
- No abstraction soup
